DEFINICION

HTML5 es la quinta versión del lenguaje de marcado HTML (HyperText Markup Language), utilizado para la creación y estructuración de contenido en la web. Se introdujo con mejoras significativas en comparación con sus versiones anteriores, incluyendo nuevas etiquetas semánticas, compatibilidad con multimedia sin necesidad de complementos externos como Flash, mejor soporte para gráficos y animaciones con Canvas y SVG, y una mayor integración con CSS y JavaScript para el desarrollo de aplicaciones web más dinámicas e interactivas.

BENEFICIOS

los beneficios de usar etiquetas:

  • Mayor compatibilidad: Funciona en todos los navegadores modernos sin necesidad de complementos adicionales.
  • Mejor accesibilidad: Incluye etiquetas semánticas que mejoran la estructura del contenido, lo que facilita la indexación por motores de búsqueda y mejora la accesibilidad para personas con discapacidades.
  • Multimedia integrada: Permite el uso de audio y video sin necesidad de complementos como Flash.

ELEMENTOS

Los elementos de HTML5 son etiquetas que estructuran y organizan el contenido de una página web, mejorando la accesibilidad, la semántica y la interacción con los usuarios.

  • <header>Define un encabezado.
  • <nav>Contiene enlaces de navegación.
  • <main>Ayuda a mejorar la accesibilidad y la semántica del sitio.
  • <aside>Se usa para mostrar información complementaria sin interferir con el contenido principal.
  • <footer>Define el pie de página con información relevante.
  • <section>Agrupa contenido relacionado en una sección
  • <article>Representa un artículo independiente dentro de la página.
  • <figure>Suele acompañarse de la etiqueta <figcaption> para agregar una descripción.
