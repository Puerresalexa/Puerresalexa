CURSO DE DISEÑO WEB

HTML 5

HTML5 es la quinta versión del lenguaje de marcado HTML (HyperText Markup Language), utilizado para la creación y estructuración de contenido en la web. Se introdujo con mejoras significativas en comparación con sus versiones anteriores, incluyendo nuevas etiquetas semánticas, compatibilidad con multimedia sin necesidad de complementos externos como Flash, mejor soporte para gráficos y animaciones con Canvas y SVG, y una mayor integración con CSS y JavaScript para el desarrollo de aplicaciones web más dinámicas e interactivas.

Caracteristicas:

  • Etiquetas semánticas:mejoran la estructura y accesibilidad del contenido, facilitando su interpretación por buscadores y lectores de pantalla.
  • Multimedia sin plugins:Permite integrar contenido de audio y video de manera directa
  • Canvas y SVG:Ofrece soporte para gráficos dinámicos y animaciones con las tecnologías Canvas y SVG.
  • Almacenamiento local:Integra capacidades como localStorage y sessionStorage, ideales para almacenar datos de manera temporal o permanente en el navegador.

CSS

CSS (Cascading Style Sheets, o Hojas de Estilo en Cascada) es un lenguaje utilizado para definir el diseño y la presentación de documentos HTML. Su propósito principal es mejorar la apariencia de las páginas web, permitiendo la personalización de elementos como colores, tipografías, tamaños, márgenes, alineaciones y más.

Aquí te menciono algunas de las más importantes:

  • Separación de contenido y diseño:Mantiene el HTML centrado en la estructura, mientras CSS se ocupa del estilo, facilitando la administración del código.
  • Estilos en cascada:Las reglas pueden heredarse o ser sobrescritas, lo que da flexibilidad al diseño.
  • Compatibilidad con múltiples dispositivos:Permite crear sitios responsivos, adaptándose a distintos tamaños de pantalla.
  • Selectores y clases avanzadas:Facilita la personalización de elementos específicos de la página sin modificar el HTML.
  • Animaciones y transiciones:Agrega movimiento sin necesidad de JavaScript, mejorando la experiencia del usuario.
  • Uso de variables:Permite definir valores reutilizables para mantener un diseño más consistente y fácil de modificar
  • Preprocesadores como SASS y LESS:Extienden sus funcionalidades, facilitando la creación de estilos más organizados y dinámicos

JavaScript

JavaScript es un lenguaje de programación de alto nivel que permite agregar interactividad y dinamismo a las páginas web. Es uno de los pilares fundamentales del desarrollo web, junto con HTML y CSS.

Algunas características clave de JavaScript:

  • Lenguaje interpretado:No necesita compilación, ya que los navegadores lo ejecutan directamente.
  • Versátil y potente:Se usa tanto en el frontend (para mejorar la experiencia del usuario) como en el backend (con entornos como Node.js).
  • Eventos y manipulación del DOM:Permite modificar elementos HTML en tiempo real, respondiendo a eventos como clics y movimientos del mouse.
  • Programación asincrónica:Facilita la carga eficiente de contenido con técnicas como promesas y async/await.
